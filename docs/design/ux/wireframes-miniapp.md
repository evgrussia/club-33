# Wireframes Mini-app (Telegram WebApp)

Стек: React 18 + Vite 6 + TelegramUI (DEC-006). Тон — на «ты». Все экраны учитывают `themeParams` (light/dark), MainButton, BackButton, Haptic.

Легенда:
- `[MainButton]` — нативная кнопка Telegram внизу экрана
- `[BackButton]` — нативная стрелка назад
- `(haptic.X)` — событие haptic feedback
- `(loading)` / `(error)` / `(empty)` — состояния

## UI-feedback механизмы (требование Analytics)

| Где | Что собираем | Как |
|---|---|---|
| Match results | `match_quality` 1–5 + label «полезно/нейтрально/мимо» | Через 1 час — push в бот + экран feedback в mini-app |
| Ask answer | `kb_usefulness` бинарно «Полезно / Так себе» | Сразу под ответом |
| Notification preferences | Подписка на digest on/off | Профиль → Настройки уведомлений |

Без жёсткого downvote (DEC-R-005): везде минимум 3-х значная шкала с нейтральным средним.
