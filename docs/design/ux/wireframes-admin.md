# Wireframes Web-админки

Стек: React 18 + Vite 6. Десктоп-first (1280+), responsive до tablet. Роли: **super / mod / support** (DEC-005).

Шаблон layout:

```
┌─────────────────────────────────────────────────────────────┐
│ TOP BAR: [Search Cmd+K]  ····  [🔔]  [👤 admin@club33 ▾]   │
├──────────┬──────────────────────────────────────────────────┤
│ SIDEBAR  │  BREADCRUMB › Section › Subpage                  │
│          │                                                   │
│ 📊 Dash  │  ┌────────────────────────────────────────────┐  │
│ 👥 Users │  │                                            │  │
│ 📩 Apps  │  │             CONTENT AREA                   │  │
│ 📅 Intv  │  │                                            │  │
│ 🎭 Roles │  │                                            │  │
│ 🚨 Compl │  │                                            │  │
│ 💰 Fin   │  │                                            │  │
│ 📈 Funnl │  │                                            │  │
│ 🤖 AI    │  │                                            │  │
│ ⚙️ Sett  │  │                                            │  │
│ 📜 Logs  │  └────────────────────────────────────────────┘  │
└──────────┴──────────────────────────────────────────────────┘
```

Пункты sidebar **скрываются** если роли не положены (а не disabled).

## 13. Экспорт CSV/Excel

Везде, где есть list-view (Пользователи, Платежи, Заявки, Воронка, Логи) — кнопка `[Экспорт ▾]`:

```
┌──────────────────────────┐
│  📥 Экспорт              │
├──────────────────────────┤
│  • CSV (текущий фильтр)  │
│  • Excel                 │
│  • CSV (без фильтра)     │
└──────────────────────────┘
```

После клика — запись в `ExportLog(actor, dataset, filter, rows)`, скачивание.
