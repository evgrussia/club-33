# Admin Notifications — Уведомления администраторам

Тон: рабочий, телеграфный. Сразу — кого, что, действие. Все сообщения уходят в админ-чат Telegram + дублируются в админке (badge / notification center).

---

## 1. Воронка кандидата

### 1.1. Новая заявка

```
📥 Новая заявка
@username — [Имя]
Источник: Google-форма
Отправлено: 15.05.2026 14:32
```
**Кнопки:**
- `Открыть в админке`
- `Одобрить`
- `Отклонить`

### 1.2. Заявка одобрена (для логирования в чат)

```
✅ Заявка одобрена
@username — [Имя]
Одобрил: [admin @name]
```
**Кнопки:** —

### 1.3. Заявка отклонена

```
❌ Заявка отклонена
@username — [Имя]
Отклонил: [admin @name]
Причина: [текст или пусто]
```
**Кнопки:** —

## 2. Платежи

### 2.1. Новая оплата (успешная)

```
💰 Оплата прошла
@username — [Имя]
Тариф: 12 месяцев
Сумма: 280 USD / 25 480 ₽
Метод: USDT TRC20
TXID: 0xabc...
```
**Кнопки:**
- `Открыть платёж`
- `Открыть профиль`

### 2.2. Проблема оплаты — > 10 минут без подтверждения

```
⚠️ Платёж не подтверждён
@username
Инвойс: #INV-00342
Метод: USDT TRC20
Создан: 14:20 (15 минут назад)
Сумма: 280 USDT
```
**Кнопки:**
- `Открыть инвойс`
- `Связаться с пользователем`

### 2.3. Неверная сумма

```
⚠️ Сумма не сходится
@username
Ожидалось: 280 USDT
Получено: 275 USDT
Разница: -5 USDT
```
**Кнопки:**
- `Принять как есть`
- `Запросить доплату`
- `Связаться`

### 2.4. Late payment (превышен expires_at)

```
🕒 Late payment review
@username
Инвойс: #INV-00342
Истёк: 14:50 (30 мин назад)
Платёж пришёл: 14:55 (после истечения)
Нужно решение: засчитывать или отклонить?
```
**Кнопки:**
- `Засчитать`
- `Вернуть платёж`
- `Открыть в админке`

### 2.5. СБП — ручная сверка

```
🏦 СБП ожидает сверки
@username
Заявленная сумма: 25 480 ₽
Назначение: Club33 @username
Время: 14:25
```
**Кнопки:**
- `Подтвердить получение`
- `Не получено`

### 2.6. Продление подписки

```
🔄 Продление
@username
Тариф: 12 месяцев
Сумма: 280 USD
Новый срок: до 15.05.2028
```
**Кнопки:**
- `Открыть профиль`

## 6. Технические уведомления

### 6.1. Ошибка webhook

```
🛑 Webhook не получен
Провайдер: ЮKassa
Инвойс: #INV-00342
Истекло: 15 мин с момента ожидания
Проверить вручную в кабинете провайдера.
```
**Кнопки:**
- `Открыть инвойс`
- `Открыть лог`

### 6.2. Бот не админ канала/чата

```
🛑 Системная ошибка
Бот не является администратором в [канал/чат].
Не сможет создавать invite-ссылки.
```
**Кнопки:**
- `Открыть runbook`

### 6.3. AI-сервис недоступен

```
⚠️ AI недоступен
Сервис: [matching / ask / digest]
Провайдер: Anthropic
Ошибка: [код / описание]
Активные запросы поставлены в очередь.
```
**Кнопки:**
- `Открыть мониторинг`

### 6.4. Превышен месячный бюджет AI

```
💸 AI бюджет превышен
Текущий расход: $XXX (план: $YYY)
Активные сервисы временно ограничены.
```
**Кнопки:**
- `Открыть дашборд`

### 6.5. Cron-задача упала

```
🛑 APScheduler — ошибка
Задача: [respect_reset / digest / subscription_reminder]
Время: 15.05.2026 03:00 МСК
Лог: [link]
```
**Кнопки:**
- `Открыть лог`
