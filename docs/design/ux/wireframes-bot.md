# Wireframes Telegram-бот

ASCII-описания ключевых dialog-флоу. Тон — на «ты», коротко. `[Кнопка]` — inline-кнопка; `(reply)` — reply-keyboard; `«фраза»` — текст бота.

## 17. Состояния FSM — соответствие экранам

| FSM | Какой экран по умолчанию показываем |
|---|---|
| applied / screening | «Заявка на модерации» + статус |
| approved | Раздел 2.1 (выбор слота интервью) |
| awaiting_payment | Раздел 2.2 → 3 (тариф → метод) |
| paid / awaiting_interview | Подтверждение оплаты + слот интервью |
| interview_done | Если не было approve — ожидание; если approve → закон |
| awaiting_law | Раздел 7 (закон + ввод фразы) |
| onboarding_video | Видео-приветствие + invite-ссылки (Раздел 7 нижняя часть) |
| active | Главное меню участника + «Мой доступ» |
| grace_period | Та же «Мой доступ» + push «Подписка истекла» |
| lifetime_active | Lifetime-вариант «Мой доступ» |
