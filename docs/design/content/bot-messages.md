# Bot Messages — Тексты Telegram-бота

Все тексты на «ты». Формат: `состояние/команда → текст → кнопки`.

---

## 1. Воронка кандидата

### 1.1. `/start` (новый пользователь)

```
Привет. Это бот Клуба 33 — закрытого сообщества людей дела.

Чтобы попасть в клуб, нужно пройти три шага:
1. Заполнить анкету
2. Поговорить с основателем
3. Принять закон клуба

Готов?
```

**Кнопки:**
- `Подать заявку`
- `Что за клуб?`

---

### 1.2. `Что за клуб?`

```
Клуб 33 — это сообщество, где люди помогают друг другу делом, временем и связями.

Здесь нет «нетворкинга» — есть конкретные просьбы и конкретная помощь.

Подписка платная: 6 или 12 месяцев. Вступление — через интервью с основателем.
```

**Кнопки:**
- `Подать заявку`
- `Назад`

---

### 1.3. Подача заявки (ссылка на Google-форму)

```
Открой анкету и заполни — это 5–7 минут.
После отправки вернись сюда — я подтвержу получение.
```

**Кнопки:**
- `Открыть анкету` (URL на Google-форму)
- `Заполнил, проверь`

---

### 1.4. Подтверждение отправки заявки

```
Заявка получена ✅
Команда клуба посмотрит её и ответит в течение 1–2 суток.
```

**Кнопки:** —

---

### 1.5. Уведомление об одобрении заявки

```
Анкета понравилась — двигаемся дальше.

Следующий шаг — выбор тарифа и оплата. После оплаты запишешься на интервью с основателем.
```

**Кнопки:**
- `Выбрать тариф`

---

### 1.6. Уведомление об отклонении заявки

```
Спасибо за заявку. В этот раз не получилось — но это не отказ навсегда.

Можешь подать заявку снова через 3 месяца.
```

**Кнопки:** —

## 2. Оплата

### 2.1. Экран выбора тарифа

```
Выбери срок подписки:

• 6 месяцев — 150 USD
• 12 месяцев — 280 USD (экономия ~7%)

Все цены в долларах, оплатить можно USDT, картой через ЮKassa или СБП.
```

**Кнопки:**
- `6 месяцев — 150 USD`
- `12 месяцев — 280 USD`
- `Назад`

---

### 2.2. Выбор метода оплаты

```
Тариф: [12 месяцев — 280 USD]

Чем платишь?
```

**Кнопки:**
- `USDT (TRC20)`
- `USDT (TON)`
- `Карта (ЮKassa, RUB)`
- `СБП (RUB)`
- `Назад`

---

### 2.3. Создание инвойса — USDT (TRC20/TON)

```
Тариф: [12 месяцев]
Сумма: [280 USDT TRC20]

Адрес: `TXXXxxxxxxxxxxxxxxxxxxxxxxxx`
(нажми, чтобы скопировать)

⏳ Жду подтверждения сети. Это 5–10 минут после перевода.
Инвойс действителен 30 минут.
```

**Кнопки:**
- `Копировать адрес`
- `Проверить статус`
- `Отменить`

---

### 2.4. Создание инвойса — ЮKassa

```
Тариф: [12 месяцев]
Сумма: [25 480 ₽] (курс 91.0 ₽/USD на момент инвойса)

Нажми «Оплатить» — откроется страница ЮKassa.
После оплаты вернёшься сюда, статус обновится автоматически.
```

**Кнопки:**
- `Оплатить картой`
- `Отменить`

---

### 2.5. Создание инвойса — СБП

```
Тариф: [12 месяцев]
Сумма: [25 480 ₽]

Реквизиты СБП:
• Получатель: [ФИО]
• Банк: [Тинькофф]
• Телефон: +7 XXX XXX XX XX

В назначении укажи: `Club33 @username`

После перевода нажми «Я оплатил» — мы сверим вручную в течение часа.
```

**Кнопки:**
- `Я оплатил`
- `Отменить`

---

### 2.6. Ожидание подтверждения оплаты

```
⏳ Жду подтверждения. Обычно это 5–10 минут.

Можешь закрыть бота — я напишу, как только всё пройдёт.
```

**Кнопки:**
- `Проверить статус`
- `Поддержка`

---

### 2.7. Превышен срок ожидания (>10 минут)

```
Подтверждения пока нет. Это не страшно — иногда сеть тормозит.

Если перевод сделан, напиши в поддержку — разберёмся.
```

**Кнопки:**
- `Поддержка`
- `Проверить ещё раз`

---

### 2.8. Подтверждение оплаты успешной

```
Оплата прошла ✅
Тариф: [12 месяцев]
Действует до: [15.05.2027]

Следующий шаг — интервью с основателем. Выбери удобный слот.
```

**Кнопки:**
- `Записаться на интервью`

## 3. Календарь / Интервью

### 3.1. Ссылка на календарь club33calendar

```
Выбери удобное время для разговора с основателем. Интервью займёт ~30 минут, по видеосвязи.

```

**Кнопки:**
- `Открыть календарь` (deep-link в mini-app)

---

### 3.2. Подтверждение записи

```
Записал ✅
Дата: [пт, 22 мая, 19:00 МСК]
Ссылка на встречу придёт за час до начала.
```

**Кнопки:**
- `Перенести`
- `Отменить`

---

### 3.3. Напоминание о неподтверждённой записи

```
Заметил, что ты ещё не выбрал слот для интервью.

Удобнее записаться сейчас, пока есть свободные окна.
```

**Кнопки:**
- `Открыть календарь`

---

### 3.4. Напоминание за час до интервью

```
Через час — интервью с основателем.
Ссылка: [URL]

Если планы изменились, перенеси встречу.
```

**Кнопки:**
- `Перейти к встрече`
- `Перенести`

---

### 3.5. No-show (пропущенное интервью)

```
Тебя не было на интервью. Бывает.

Запишись заново — мы дадим ещё один шанс.
```

**Кнопки:**
- `Записаться снова`
- `Поддержка`

---

### 3.6. Уведомление об одобрении после интервью

```
Основатель сказал «да». Добро пожаловать в Клуб 33.

Остался последний шаг — прочитать и принять закон клуба.
```

**Кнопки:**
- `Читать закон клуба`

## 4. Закон клуба

### 4.1. Показ закона клуба

```
Это закон Клуба 33. Прочитай внимательно — мы живём по нему.

[TBD: финальный текст закона, OD-CLUB-04/05]

Если принимаешь — введи фразу согласия следующим сообщением (ручной ввод, не кнопка).
```

**Кнопки:**
- `Открыть полный текст` (если длинный — выводим из БД)

---

### 4.2. Запрос фразы согласия

```
Введи одну из фраз согласия (скопируй и отправь):

• «Я принимаю закон Клуба 33 и согласен с его правилами.»
• «Закон клуба прочитал, принимаю, готов вступать.»

Или напиши свой вариант — короче и образнее.
```

**Кнопки:** —

---

### 4.3. Подтверждение принятия закона

```
Принято. Теперь ты в клубе ✅

Сейчас пришлю ссылки на канал и чат — они одноразовые, действуют 24 часа.
```

**Кнопки:** —

---

### 4.4. Неподходящая фраза согласия

```
Эта фраза не подходит под закон. Перечитай и попробуй сформулировать иначе — своими словами или скопируй один из вариантов.
```

**Кнопки:**
- `Показать варианты ещё раз`

## 7. «Мой доступ»

### 7.1. Команда `/access` или «Мой доступ»

```
👤 [Имя]
Роль: [Участник]
Тариф: 12 месяцев
Действует до: 15.05.2027 (осталось 247 дней)
Респекты: 12/30 в этом месяце
Дни (валюта): 5 дней

Что хочешь сделать?
```

**Кнопки:**
- `Продлить`
- `История платежей`
- `Открыть mini-app`

---

### 7.2. Напоминание о продлении (за 10 дней)

```
Через 10 дней подписка закончится. Продлим?

При продлении сейчас цена та же. После окончания — оформляем как новый период.
```

**Кнопки:**
- `Продлить на 6 мес`
- `Продлить на 12 мес`
- `Не сейчас`

---

### 7.3. Напоминание за 3 дня

```
3 дня до конца подписки. После окончания доступ к чату и каналу закроется.
```

**Кнопки:**
- `Продлить`

---

### 7.4. Доступ закончился

```
Подписка закончилась. Доступ к чату и каналу закрыт.

Если хочешь продолжить — оформи новый период. История и репутация сохранены.
```

**Кнопки:**
- `Продлить доступ`
