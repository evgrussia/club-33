# Коммерческое предложение

## «Клуб 33» — Telegram-бот, mini-app, web-админка, AI-сервисы

---

## 1. Резюме

Разработка закрытой клубной платформы на базе Telegram: бот для воронки и оплат, mini-app для социального слоя (респекты, дары, профиль, матчинг), web-админка для управления и AI-сервисы на Anthropic Claude (матчинг, база знаний, еженедельный пульс).

**3 фазы релиза** с независимыми поставками в production:
- **Фаза 1 (MVP):** базовый бот, оплаты, доступ, минимальная админка — ≈ **420 чч**
- **Фаза 2 (Социал + AI):** роли, респекты, звёзды, жалобы, матчинг, RAG, digest — ≈ **365 чч**
- **Фаза 3 (Экономика времени):** дар, сжигание, lifetime-бюджет, конвертер — ≈ **110 чч**

**Сквозные работы** (инфра, CI/CD, тестирование, дизайн, PM): ≈ **210 чч**

### Итого: **~ 1 105 человеко-часов** на весь продукт

Команда из 3–4 человек (backend, frontend, AI/bot, devops/QA part-time) → ориентировочно **2 месяца** до полного готового продукта при параллельной работе.

**MVP в production** — после Фазы 1, ориентир **3–4 недели** от старта.

---

```
Неделя 1-3:   Фаза 1 — MVP                      [бот + оплаты + доступ + админка]

```

## 8. Допущения и риски

### Что включено в оценку
- Разработка кода + миграции + тесты + код-ревью
- Базовая инфраструктура и деплой на production
- Логирование, мониторинг, основные runbooks
- Документация для администраторов
- 1 demo для заказчика после каждой фазы
- 2 итерации правок после demo по каждой фазе (≈ 5-10% от фазы)

### Что НЕ включено (отдельно по запросу)
- Юридическое сопровождение (закон клуба, privacy policy) — заказчик находит юриста
- Маркетинг и контент (статьи, тексты для канала клуба)
- Self-service KYC через провайдеров (на старте — ставится вручную)
- Оплата зарубежной картой (архитектура заложена, реализация — отдельно ~16-24 чч)
- Реферальная программа (Фаза 4+, ~32-48 чч)
- Маркетплейс услуг (Фаза 4+, ~80-120 чч)
- Расширенная аналитика (например, BI-дашборд Metabase) — отдельно ~16 чч
- Поддержка 24/7 (только рабочее время команды)
- Локализация на другие языки

### Ключевые риски и зависимости
| Риск | Влияние | Митигация |
|---|---|---|
| Юр. согласование закона клуба занимает > 2 недель | Сдвиг production-deploy | Параллельно с разработкой; production за юр.-стопом |
| AI стоимость >$2 на участника/мес | Превышение бюджета на inference | Cost-caps и переключение на Haiku |
| USDT-провайдер (Cryptomus) откажет | Переключение на NowPayments | Adapter pattern — ~1 неделя на switch |
| Открытые вопросы клуба (звёзды, неактивность) | Задержка калибровки в Фазе 2 | Конфигурируемые параметры в админке |
| Listener-бот падает / Telegram банит | Часть KB не индексируется | Мониторинг + auto-restart + резервный аккаунт |

### Оплата и этапность
- Предлагаемая модель: фиксированная стоимость за фазу с раздельным актом
- Аванс 30% на старте каждой фазы, 40% после demo, 30% после стабилизации
- Гарантия 30 дней после сдачи каждой фазы (исправление багов бесплатно)
